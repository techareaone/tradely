# PyX - Python to EXE Conversion Wizard for Windows
## Built in python.
*Stage = Prototype <br> Documentation = Under Development*

The objective of this project is to create a simple to use multi-step wizard to convert python to exe, which can easily be shared with other users. This will be optimised to work with TKinter and CustomTKinter, as that is the most applicable Python GUI Tool for developing bots. In future updates it may be optimised to work with other tools.
